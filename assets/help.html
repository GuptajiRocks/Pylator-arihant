Welcome to Pylator!

Pylator is a versatile program designed to assist with a wide variety of calculations, ranging from basic arithmetic to scientific computations. This help guide will walk you through each feature and mode so that you can make the most out of the tool.

Note: Make sure to enter valid inputs for each mode to avoid calculation errors.

Modes Overview

  • Calculator Mode: Perform basic arithmetic and trigonometric calculations.
  • Quadratic Solver: Solve quadratic equations using real and complex numbers.
  • BMI Calculator: Calculate your Body Mass Index (BMI) using your height and weight.
  • Equation Solver: Solve two-variable linear equations.
  • Electric Circuit Solver: Calculate values such as voltage, current, resistance, and power using Ohm's law.
  • Right-Angle Triangle Solver: Solve for unknown sides of a right-angled triangle.
  • Marathon Calculator: Compute motion properties such as horizontal and vertical distances for inclined projections.

How to Use Each Mode

1. Calculator Mode

In this mode, you can perform standard arithmetic operations like addition, subtraction, multiplication, and division. It also supports trigonometric functions such as sine, cosine, and tangent.

  • Enter the expression you want to calculate using the buttons on the screen.
  • Press = to see the result.
  • Use C to clear the screen, and DEL to delete the last entry.

2. Quadratic Solver

Use this mode to solve quadratic equations of the form ax² + bx + c = 0.

  • Input the values of a, b, and c.
  • The program will calculate the roots of the equation and display them.
  • If the discriminant is negative, the program will inform you that the roots are complex.

3. BMI Calculator

The BMI calculator allows you to check your Body Mass Index based on your height (in meters) and weight (in kilograms).

  • Enter your height and weight in the respective fields.
  • The program will calculate your BMI and categorize it as Underweight, Normal Weight, Overweight, or Obese.

4. Equation Solver

This mode allows you to solve two-variable linear equations in the form ax + by = c.

  • Input the coefficients for two equations.
  • The program will calculate and display the values of x and y.

5. Electric Circuit Solver

This mode allows you to calculate electric circuit parameters based on Ohm’s law: V = IR.

  • Enter two of the three values: Voltage (V), Current (I), or Resistance (R).
  • The missing value will be calculated, and the program will also compute the power (Wattage).
  • A graph of the voltage-current relationship can be generated using the "Graph" button.

6. Right-Angle Triangle Solver

Use this mode to calculate the sides of a right-angled triangle based on the Pythagorean theorem.

  • Enter two known sides (either legs or one leg and the hypotenuse).
  • The program will compute and display the unknown side.

7. Marathon Calculator

This mode allows you to compute the horizontal or vertical distance for an object launched at a specific angle.

  • Input the angle of projection and other relevant parameters.
  • The program will compute and display the motion results based on trigonometric principles.

Tips for Optimal Use

  • Ensure that your inputs are valid. For instance, height in the BMI calculator should be in meters and weight in kilograms.
  • Switch between modes using the buttons on the left sidebar. The display will update accordingly.
  • If an error occurs, such as "complex roots" in the quadratic solver, make sure to check the discriminant for negative values.
